
# Cat-Fish

A simple **PC** web game where you control a cat to get as many fish as possible in a short amount of time. There are collectibles, which can increase, decrease or just straight up freeze your time. There are 12 cats to choose from - each with it's own abilities.


## Authors

- [wikt0r3k](https://www.github.com/realwikt0r3k) - Lead and only developer, soundtrack
- [AnKaa](https://www.instagram.com/kocankaa) - UI, Cat Sprites, *overall graphic design*
- [Vgrechisher](https://www.youtube.com/@vgrechi)) - SFX


## What this project contains

- **Menus**. Lots of menus.
- **Shop system**, where you can either buy a new cat, or try to get one from buying a cat in a bag.
- **Collectibles**, which at the end of the round convert into additional coins.
- **Cat abilities** - more fish in the game, faster cat, smaller cat, less time, more time etc.
- **Different fish variants** - they can give you points, they can deduct them, maybe give you time or just freeze it entirely.

## What this project might contain

- **Mobile support** - this game in it's early stage had a mobile support, which was just broken, which is why at this stage of production we decided to **remove** mobile support. It will be re-added in the future.
- **More themes** - as of right now, Cat-Fish has only 1 theme, which is the default one.
- **Rivalry** - Cat-Fish may not be an online game *yet*, but we're aiming for something in the game, which you can compete in with your friends.
- **Better code structure** - before we remade the game, it's code was a 1k+ line spaghetti. Right now we *(me)* are trying to refactor the code, which will make it easier to modify or add things to the game.
- **Better graphics** - we know it's not that hard to make a game with decent graphics, but it's also not easy, when there's only one person, who actually draws them.
